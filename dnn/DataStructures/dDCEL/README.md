# dDCEL v1.0

## What is 'dDCEL'
dDCEL is a basic data structure to store geometric objects in high dimensional space. dDCEL consists of several classes. To see the manual of each classes, check out *\<classname\>.md* from this folder.

## Classes
[dDCEL](dDCEL.md)

## Example
Supports 3d visualization using OpenGL. Two examples are included, dodecahedron and tetrahedron.
